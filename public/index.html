/api/v1/search

API URL: https://treasurejsapi.herokuapp.com/api/v1/search?find=as&size=1

This API will search the name of library which you will send in the find parameter

Query Parameters

Datatype Parameter Description
String find string to search
Integer size size of response list
Boolean count get count of response only

/api/v2/search

API URL: https://treasurejsapi.herokuapp.com/api/v2/search?find=as&size=1&exact=false&projection=["name"]

this api will check the name from the start

Query Parameters

Datatype Parameter Description
String find string to search
Integer size We can restrict the response size using this
Boolean exact We can get exact match response
Boolean count get count of response only
Array projection We can restrict the values we need from the API response. You can see the values you can use with projection to work here.
