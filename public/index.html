/api/v2/search/description

    {
        "find": "MobX"
    }

case insensitive

/api/v2/search/docs

    {
        "find": "git"
    }

/api/v2/search/website

    {
        "find": "git"
    }

/api/v2/search/github

    {
        "find": "git"
    }

Extra Parameters

We can restrict the response size using

    {
        "size": 5
    }

We can use exact boolean flag to get exact match response

    {
        "exact": true
    }

We can add the projection i.e keys we need in the response

    {
        "projection": ["name"]
    }

You can only add the below fields in projection array

  1. ""
  2. "_id"
  3. "name"
  4. "description"
  5. "github"
  6. "website"
  7. "docs"
  8. "other"
