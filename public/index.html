APIs Version 1 Endpoints

/api/v1/search

API URL: https://treasurejsapi.herokuapp.com/api/v1/search? find=as&size=1

This API will search the name of library which you will send in the find parameter

Query Parameters

  • find - string to search
  • size - size of response list

APIs Version 2 Endpoints

/api/v2/search

API URL: https://treasurejsapi.herokuapp.com/api/v2/search? find=as&size=1&exact=false&projection=["name"]

this api will check the name from the start

Query Parameters

  • find - string to search
  • size - We can restrict the response size using this
  • exact - We can use exact boolean flag to get exact match response
  • projection - We can restrict the values we need from the API response. You can see the values you can use with projection to work here.
